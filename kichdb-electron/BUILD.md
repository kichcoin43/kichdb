# Инструкция по сборке KICH DB

## Требования

- Node.js 18+ (рекомендуется 20)
- npm или pnpm

## Шаги сборки

### 1. Скачать проект

Скопируйте папку `kichdb-electron` на свой компьютер.

### 2. Установить зависимости

```bash
cd kichdb-electron
npm install
```

### 3. Собрать EXE

```bash
# Для Windows
npm run dist:win

# Для Linux
npm run dist:linux

# Для macOS
npm run dist:mac
```

### 4. Результат

После сборки в папке `release` появятся готовые файлы:

**Windows:**
- `KICHDB-1.0.0-portable.exe` - портативная версия (запускается сразу, не требует установки)
- `KICHDB Setup 1.0.0.exe` - установщик

**Linux:**
- `KICHDB-1.0.0.AppImage`
- `kichdb_1.0.0_amd64.deb`

**macOS:**
- `KICHDB-1.0.0.dmg`

## Использование

### Портативная версия (рекомендуется)

1. Скопируйте `KICHDB-1.0.0-portable.exe` в любую папку
2. Запустите его
3. Рядом с exe создадутся:
   - `database.json` - все данные
   - `storage/` - загруженные файлы

### Установщик

1. Запустите `KICHDB Setup 1.0.0.exe`
2. Выберите папку установки
3. Запустите из меню "Пуск" или ярлыка

## Решение проблем

### Ошибка "npm install"

Убедитесь, что Node.js установлен правильно:
```bash
node -v  # должно показать версию 18+
npm -v   # должно показать версию 9+
```

### Ошибка при сборке

Попробуйте очистить кэш:
```bash
rm -rf node_modules
rm package-lock.json
npm install
npm run dist:win
```

### Antivirus блокирует exe

Добавьте exe в исключения антивируса. Это нормально для свежесобранных exe.

## Обновление

Для обновления приложения:
1. Соберите новую версию
2. Замените exe на новый
3. Файлы `database.json` и `storage/` сохраняются

## Настройка

Изменить иконку приложения:
1. Положите файл `icon.ico` (256x256) в папку `build/`
2. Пересоберите приложение

Изменить название:
1. Отредактируйте `package.json`:
   - `name` - техническое имя
   - `productName` в секции `build` - отображаемое имя
